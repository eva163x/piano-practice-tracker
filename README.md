### Installation

1. Clone the repo

`git clone https://github.com/your-username/piano-practice-tracker.git`
`cd piano-practice-tracker`

2. Install dependencies

`npm install`

3. Start the server

`npm start`

4. Navigate to localhost:8000 in your browser
